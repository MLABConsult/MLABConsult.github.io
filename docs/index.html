Sobre

Mas afinal, quem somos?.

  • O que nos motiva?

    Nosso propósito é romper as barreiras entre problema e solução de negócio. Para isso utilizamos dados e inteligência artificial.

  • Quem somos?

    Somos profissionais com vasta experiência em modelagem matemática, processamento e engenharia de dados, e engenharia de machine learning, com vontade de construir soluções para os mais diversos negócios.

  • Tecnologia

    Dentro do nosso arsenal de ferramentas para trabalhar com dados, oferecemos todo o pipelinede dados. Desde a extração dos dados e seu armazenamento, até a modelagem. Fazemos a limpeza necessária para criar o modelo que melhor se adequa a seu negócio.

  • Ferramentas

    Trabalhamos com Spark, MongoDB, SQL, Python, R, Docker e Kubernetes, MapReduce, DataLakes, Google Cloud, AWS Cloud, Azure Cloud, entre outros.
